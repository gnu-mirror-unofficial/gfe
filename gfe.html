gfe has been decommissioned, since the free software program FontForge has more advanced bitmap editing facilities. This page remains for the sake of history.

What is GNU Font Editor

GNU Font Editor (GFE) is a graphical font editor based on the GIMP Toolkit. It is easy to use and will eventually support many font types. Currently it supports only BDF font files, that can be converted to many other formats easily.

Work is now in progress to support PCF font files.

Downloading GFE

Currently, the GNU Font Editor is only available via CVS.

Development

  • To obtain the latest source code from the CVS, use the commands 
    cvs -d :pserver:anoncvs@subversions.gnu.org:/cvs login
Password: [just hit enter]
cvs -d :pserver:anoncvs@subversions.gnu.org:/cvs checkout gfe
  • You can also browse the GNU CVS repository via the CVSweb interface. Select the module gfe.
  • The development mailing list discussions are archived here.
  • To receive an announcement whenever a new version of gfe is available, send an email to info-gfe-request@gnu.org with the subject line subscribe. This is a moderated mailing list with traffic limited only to announcements.

Contributing

Send bug reports to bug-gfe@gnu.org, containing the problem encountered, the version of GFE, and error messages if any were displayed.

Subscribe to our mailing list by sending an email to bug-gfe-request@gnu.org with the subject line subscribe.
